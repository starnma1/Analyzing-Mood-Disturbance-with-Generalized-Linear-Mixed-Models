# Analyzing-Mood-Disturbance-with-Generalized-Linear-Mixed-Models

This project models the effect of nature exposure on mood using data from the TREE study. Participants completed a Total Mood Disturbance (TMD) assessment before and after spending time in either a green or urban environment. A linear mixed model was fit using glmmTMB, with random effects accounting for repeated measurements nested within participants and session dates. Fixed effects selection across 21 candidate models identified an Environment × Time interaction as the key driver of TMD change, with precipitation, age, and sex included as covariates.

Data is private.
